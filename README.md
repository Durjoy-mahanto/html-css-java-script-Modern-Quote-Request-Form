# 📋 Modern Quote Request Form

A clean, modern, and fully responsive **Quote Request Form** built using **HTML5, CSS3, and JavaScript**. This project provides a professional user interface where users can submit quote requests by entering their personal information, selecting a service, and describing their project.

## 🚀 Getting Started

### Clone the Repository

```bash
git clone https://github.com/Durjoy-Mahanto/html-css-java-script-Modern-Quote-Request-Form.git
```

### Open the Project

```bash
cd html-css-java-script-Modern-Quote-Request-Form
```

Open **index.html** in your favorite browser.

## 💡 How It Works

- Users enter their personal information.
- Select a required service.
- Add project details.
- JavaScript validates the form.
- The form is then ready for further backend integration.
